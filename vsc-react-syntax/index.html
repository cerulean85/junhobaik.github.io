WebStorm을 쓰다가 Visual Studio Code를 사용하니 React를 쓰기에 불편함이 한두가지가 아니다.

제일 처음 맞은 불편함이 JSX에서 emmet 자동완성이 작동 안되는 것인데,
해결 방법으로 여러 방법이 있었지만 제일 간단한 방법은 아래와 같다.

기본 설정에서 아래 코드를 추가해준다.

    "files.associations": {
        "*.js": "javascriptreact"
    }
}
